Caching Strategy for Containerized Repricing

Version: 1.0
Last Updated: January 16, 2026
Author: Andy Bratton
Status: Phase 1 Enhancement

Overview

This document details how caching can be implemented to eliminate database polling during claim repricing. The key insight is that caching can be implemented at the .NET wrapper layer without modifying the native C++ DLLs.

Why This Works

The native pricing DLLs don’t directly query the database during pricing. Instead, they: 1. DRG Grouper - Uses embedded data files (.dat, .bin) already in the container 2. Medicare Pricers - Receive rate data via initialization calls 3. TAC Engine - Receives contract/term data as parameters

The .NET wrapper layer can pre-load and cache all reference data, then inject it into the native DLLs via the bridge interface.

1. Current vs. Cached Architecture

Current State (Database Polling)

Sequence (Database hit on EVERY claim):

Step From To Action
1 Pricing Worker Native Bridge Price claim
2 Native Bridge SQL Server Query DRG weights
3 SQL Server Native Bridge Return DRG weights
4 Native Bridge SQL Server Query provider rates
5 SQL Server Native Bridge Return provider rates
6 Native Bridge SQL Server Query fee schedules
7 SQL Server Native Bridge Return fee schedules
8 Native Bridge Pricing DLLs Initialize with data
9 Pricing DLLs Pricing DLLs Group & price
10 Pricing DLLs Native Bridge Result
11 Native Bridge Pricing Worker Result

Problem: Database hit on EVERY claim

Target State (Cached)

On Container Startup:

Step From To Action
1 Pricing Worker Azure Redis Cache Load reference data
2 Redis Cache Pricing Worker Cached data (or miss → DB)
3 Pricing Worker In-Memory Cache Populate local cache

On Each Claim (Zero DB Hits):

Step From To Action
1 Pricing Worker In-Memory Cache Get provider rates
2 In-Memory Cache Pricing Worker Cached data
3 Pricing Worker Native Bridge Price claim + cached data
4 Native Bridge Pricing DLLs Initialize with data
5 Pricing DLLs Pricing DLLs Group & price
6 Pricing DLLs Native Bridge Result
7 Native Bridge Pricing Worker Result

Result: Zero database queries per claim

2. Caching Layers

Layer 1: Container-Embedded Data (Static)

Data that never changes during container lifetime - baked into the image:

Data Size Refresh Cycle Update Method
DRG Grouper files (.dat, .bin) ~150 MB Annually Container rebuild
ICD-10 code reference ~18 MB Annually Container rebuild
CPT/HCPCS reference ~2 MB Annually Container rebuild
Revenue code reference ~50 KB Annually Container rebuild

No caching needed - data is already in container filesystem.

Layer 2: In-Memory Cache (Per Container)

Data loaded at container startup and held in memory:

Data Size TTL Source
DRG Weights (all versions) ~15 MB 24 hours Redis → SQL
APC Rates (current quarters) ~10 MB 6 hours Redis → SQL
RVU Tables (current year) ~25 MB 24 hours Redis → SQL
GPCI Tables ~500 KB 24 hours Redis → SQL
Conversion Factors ~1 KB 24 hours Redis → SQL

Layer 3: Redis Distributed Cache (Shared)

Data shared across all container instances:

Data Size TTL Pattern
Provider Rates (by provider) ~5 MB total 1 hour On-demand with LRU
Fee Schedules (by state) ~26 MB total 6 hours Pre-loaded
Contract Definitions Variable 5 minutes On-demand
Medicaid Rates (by state) ~4 MB/state 12 hours Pre-loaded

Cache Manager

// CacheManager.cs
public class CacheManager : ICacheManager, IHostedService
{
    private readonly IMemoryCache _memoryCache;
    private readonly IDistributedCache _distributedCache;
    private readonly IServiceProvider _services;
    private readonly CacheOptions _options;
    private readonly ILogger<CacheManager> _logger;
    
    public async Task StartAsync(CancellationToken ct)
    {
        _logger.LogInformation("Initializing pricing cache...");
        
        // Pre-load critical reference data at startup
        var tasks = new List<Task>
        {
            PreloadDrgWeightsAsync(ct),
            PreloadApcRatesAsync(ct),
            PreloadRvuTablesAsync(ct),
            PreloadGpciTablesAsync(ct),
            PreloadConversionFactorsAsync(ct)
        };
        
        await Task.WhenAll(tasks);
        
        _logger.LogInformation("Pricing cache initialized. Memory usage: {Size} MB",
            GC.GetTotalMemory(false) / (1024 * 1024));
    }
    
    private async Task PreloadDrgWeightsAsync(CancellationToken ct)
    {
        // Try distributed cache first
        var cacheKey = "drg-weights:all";
        var cached = await _distributedCache.GetAsync(cacheKey, ct);
        
        if (cached != null)
        {
            var weights = DeserializeDrgWeights(cached);
            _memoryCache.Set(cacheKey, weights, _options.DrgWeightsTtl);
            _logger.LogInformation("DRG weights loaded from Redis: {Count} entries", 
                weights.Count);
            return;
        }
        
        // Cache miss - load from database
        using var scope = _services.CreateScope();
        var provider = scope.ServiceProvider.GetRequiredService<IDrgWeightProvider>();
        var weights = await provider.LoadAllAsync(ct);
        
        // Store in distributed cache
        await _distributedCache.SetAsync(cacheKey, SerializeDrgWeights(weights),
            new DistributedCacheEntryOptions { AbsoluteExpirationRelativeToNow = TimeSpan.FromHours(24) },
            ct);
        
        // Store in memory cache
        _memoryCache.Set(cacheKey, weights, _options.DrgWeightsTtl);
        
        _logger.LogInformation("DRG weights loaded from database: {Count} entries", 
            weights.Count);
    }
    
    // Get with fallback pattern
    public async Task<T?> GetAsync<T>(string cacheKey, 
        Func<Task<T>> factory, 
        TimeSpan? memoryTtl = null,
        TimeSpan? distributedTtl = null,
        CancellationToken ct = default) where T : class
    {
        // L1: Check in-memory cache
        if (_memoryCache.TryGetValue<T>(cacheKey, out var memValue))
        {
            return memValue;
        }
        
        // L2: Check distributed cache
        var distValue = await _distributedCache.GetAsync(cacheKey, ct);
        if (distValue != null)
        {
            var value = Deserialize<T>(distValue);
            _memoryCache.Set(cacheKey, value, memoryTtl ?? _options.DefaultMemoryTtl);
            return value;
        }
        
        // L3: Load from source
        var loaded = await factory();
        if (loaded != null)
        {
            // Store in L2
            await _distributedCache.SetAsync(cacheKey, Serialize(loaded),
                new DistributedCacheEntryOptions 
                { 
                    AbsoluteExpirationRelativeToNow = distributedTtl ?? _options.DefaultDistributedTtl 
                },
                ct);
            
            // Store in L1
            _memoryCache.Set(cacheKey, loaded, memoryTtl ?? _options.DefaultMemoryTtl);
        }
        
        return loaded;
    }
}

4. Data Provider Implementations

DRG Weight Provider

// DrgWeightProvider.cs
public class DrgWeightProvider : IDrgWeightProvider
{
    private readonly ICacheManager _cache;
    private readonly IDbConnection _db;
    
    public async Task<decimal?> GetWeightAsync(string drgCode, string version, CancellationToken ct)
    {
        var key = $"drg:{version}:{drgCode}";
        
        return await _cache.GetAsync<DrgWeight>(
            key,
            async () => await LoadFromDatabaseAsync(drgCode, version, ct),
            memoryTtl: TimeSpan.FromHours(24),
            distributedTtl: TimeSpan.FromHours(48),
            ct
        );
    }
    
    public async Task<Dictionary<string, DrgWeight>> LoadAllAsync(CancellationToken ct)
    {
        // Load all current DRG versions in one query
        var sql = @"
            SELECT DRG_CODE, DRG_VERSION, WEIGHT, MDC, TYPE, DESCRIPTION
            FROM MCRFEDRG 
            WHERE DRG_VERSION IN ('v42.0', 'v43.0')";
        
        var results = await _db.QueryAsync<DrgWeight>(sql);
        return results.ToDictionary(d => $"{d.Version}:{d.Code}");
    }
}

Provider Rate Provider (On-Demand Cache)

// ProviderRateProvider.cs
public class ProviderRateProvider : IProviderRateProvider
{
    private readonly ICacheManager _cache;
    private readonly IDbConnection _db;
    
    public async Task<ProviderRates?> GetRatesAsync(
        string providerId, 
        int fiscalYear,
        CancellationToken ct)
    {
        var key = $"provider-rates:{providerId}:{fiscalYear}";
        
        // Shorter TTL for provider-specific data (may change)
        return await _cache.GetAsync<ProviderRates>(
            key,
            async () => await LoadFromDatabaseAsync(providerId, fiscalYear, ct),
            memoryTtl: TimeSpan.FromMinutes(30),
            distributedTtl: TimeSpan.FromHours(1),
            ct
        );
    }
    
    private async Task<ProviderRates?> LoadFromDatabaseAsync(
        string providerId, 
        int fiscalYear,
        CancellationToken ct)
    {
        var sql = @"
            SELECT PROVIDER_ID, FISCAL_YEAR, OPERATING_RATE, CAPITAL_RATE,
                   IME_FACTOR, DSH_FACTOR, LABOR_SHARE
            FROM MCRIPHSPRATES 
            WHERE PROVIDER_ID = @ProviderId AND FISCAL_YEAR = @FiscalYear";
        
        return await _db.QueryFirstOrDefaultAsync<ProviderRates>(
            sql, 
            new { ProviderId = providerId, FiscalYear = fiscalYear });
    }
}

// NativePricingBridge.cs
public class NativePricingBridge : IPricingBridge, IDisposable
{
    private readonly ICacheManager _cache;
    private bool _initialized = false;
    
    public async Task InitializeAsync(CancellationToken ct)
    {
        // Load all reference data from cache
        var drgWeights = await _cache.GetAllDrgWeightsAsync(ct);
        var apcRates = await _cache.GetAllApcRatesAsync(ct);
        
        // Inject into native layer
        var result = NativeMethods.InitializePricingContext(
            JsonSerializer.Serialize(drgWeights),
            JsonSerializer.Serialize(apcRates),
            null  // Provider rates loaded per-claim
        );
        
        if (result != 0)
            throw new PricingBridgeException($"Initialization failed: {result}");
        
        _initialized = true;
    }
    
    public async Task<PricingResult> PriceAsync(
        ClaimInput claim, 
        CancellationToken ct)
    {
        // Load claim-specific cached data
        var providerRates = await _cache.GetProviderRatesAsync(
            claim.ProviderId, 
            claim.FiscalYear, 
            ct);
        
        var contract = await _cache.GetContractAsync(
            claim.ContractId, 
            ct);
        
        var feeSchedule = await _cache.GetFeeScheduleAsync(
            claim.FeeScheduleId,
            ct);
        
        // Call native pricer with all data injected
        var result = NativeMethods.PriceEpisodeWithContext(
            JsonSerializer.Serialize(claim),
            JsonSerializer.Serialize(contract),
            JsonSerializer.Serialize(feeSchedule),
            out IntPtr resultPtr,
            out uint resultLength
        );
        
        // Parse result...
    }
}

6. Cache Warming Strategy

Container Startup Sequence

Startup Flow:

Step Action Details
1 Container Start → CacheManager StartAsync()
2 Parallel Pre-load
CacheManager → Redis Get DRG weights (Hit/Miss)
CacheManager → Redis Get APC rates (Hit/Miss)
CacheManager → Redis Get RVU tables (Hit/Miss)
3 If Cache Miss
CacheManager → SQL Database Query reference data
SQL Database → CacheManager Return data
CacheManager → Redis Store in cache
4 CacheManager Load into memory
5 CacheManager → Container Ready
6 Container → Health Check Mark Ready
7 Health Check → Container Accept traffic

Pre-warming Script

// CacheWarmingService.cs
public class CacheWarmingService : BackgroundService
{
    protected override async Task ExecuteAsync(CancellationToken ct)
    {
        var sw = Stopwatch.StartNew();
        
        _logger.LogInformation("Starting cache warming...");
        
        // Critical path - must complete before accepting traffic
        await WarmCriticalCacheAsync(ct);
        
        _logger.LogInformation("Critical cache warmed in {Elapsed}ms", 
            sw.ElapsedMilliseconds);
        
        // Mark container as ready
        _readinessProbe.MarkReady();
        
        // Background - non-blocking population
        _ = WarmBackgroundCacheAsync(ct);
    }
    
    private async Task WarmCriticalCacheAsync(CancellationToken ct)
    {
        await Task.WhenAll(
            _cache.PreloadDrgWeightsAsync(ct),
            _cache.PreloadApcRatesAsync(ct),
            _cache.PreloadRvuTablesAsync(ct),
            _cache.PreloadGpciAsync(ct)
        );
    }
    
    private async Task WarmBackgroundCacheAsync(CancellationToken ct)
    {
        // Load state Medicaid data in background
        var states = new[] { "AZ", "IL", "IN", "NY", "OH", "PA", "SC", "TX" };
        
        foreach (var state in states)
        {
            await _cache.PreloadMedicaidRatesAsync(state, ct);
            await Task.Delay(100, ct); // Throttle DB load
        }
    }
}

// CacheInvalidationHandler.cs
public class CacheInvalidationHandler
{
    private readonly IDistributedCache _cache;
    private readonly IHubContext<CacheHub> _hubContext;
    
    public async Task HandleContractChangedAsync(ContractChangedEvent evt)
    {
        // Invalidate specific contract in Redis
        var key = $"contract:{evt.ContractId}";
        await _cache.RemoveAsync(key);
        
        // Notify all container instances
        await _hubContext.Clients.All.SendAsync("InvalidateCache", new
        {
            Type = "contract",
            Key = key
        });
    }
    
    public async Task HandleAnnualRateUpdateAsync(AnnualRateUpdateEvent evt)
    {
        // Invalidate all DRG weights for new fiscal year
        var pattern = $"drg:{evt.NewVersion}:*";
        await InvalidateByPatternAsync(pattern);
        
        // Trigger re-warm
        await _cacheWarmer.WarmDrgWeightsAsync(evt.NewVersion);
    }
}

SignalR for Real-Time Invalidation

// In PricingWorker
public class CacheInvalidationListener : BackgroundService
{
    private readonly HubConnection _hubConnection;
    private readonly IMemoryCache _memoryCache;
    
    protected override async Task ExecuteAsync(CancellationToken ct)
    {
        _hubConnection.On<InvalidationMessage>("InvalidateCache", msg =>
        {
            _logger.LogInformation("Cache invalidation received: {Type} - {Key}",
                msg.Type, msg.Key);
            
            // Remove from local memory cache
            _memoryCache.Remove(msg.Key);
        });
        
        await _hubConnection.StartAsync(ct);
    }
}
